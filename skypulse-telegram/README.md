# SkyPulse Telegram Bot

AI-powered flight deal notification bot.

## Setup

`ash
# Install dependencies
pip install -r requirements.txt

# Set environment variable
export TELEGRAM_BOT_TOKEN=your_bot_token_here

# Run the bot
python bot.py
`

## Commands

- /start - Start the bot
- /search - Search for flights
- /subscribe - Subscribe to alerts
- /deals - View latest deals
- /alerts - Manage your alerts
- /price - Check price prediction
- /profile - View your profile
- /unsubscribe - Unsubscribe from all

## Get Bot Token

1. Open Telegram
2. Search for @BotFather
3. Send /newbot
4. Follow instructions
5. Copy token and set it as TELEGRAM_BOT_TOKEN
